§II:
Das Rechnen mit Reellen Zahlen Im folgenden werden die wichtigsten
Eigenschaften der Reellen Zahlen (ℝ) Definiert
Die Grundrechenarten Es gibt in ℝ zwei binäre Operationen
+ "plus" und
* "mal"
die folgende Eigenschaften erfüllen:
Seien a,b,c,d ∈ ℝ
Die Grundrechenregeln
Abgeschlossenheit
a + b ∈ ℝ
a * b ∈ ℝ
Kommutativität
a + b = b + a
a * b = b * a
Assoziativität
a + (b + c) = (a + b) + c
a(bc)=(ab)c

Deshalb lässt man solche Klammern
im Allgemeinen weg und lässt
den Malpunkt "*" weg.
Neutrale Elemente
Beide Operationen besitzen ein neutrales Element
0 bzw 1
a + 0 = a und
a * 1 = a
Inverse Elemente
beide Operationen sind umkehrbar
im folgenden Sinne:
für a ∈ ℝ ∃ -a ∈ ℝ sodass a + (-a) = 0
dann definiert man die Differenz:
a - b := a + (-b)
Außerdem:
für a ∈ ℝ, a ≠ 0 ∃ \(\frac{1}{a}\) (=\(a^{-1}\)) mit
\(a*\frac{1}{a}= 1\)
dann definiere man den Quotionten
\(\frac{a}{b}:= a*\frac{1}{b}\)   für b ≠ 0
Distributivgesetz
a * (b + c) = (a * b) + (a * c)
= ab + ac
"Punkt vor Strich"
Aus diesen Grundregeln lassen sich weitere
Regeln ableiten
Bruchrechenregeln:
\(\frac{a}{b} + \frac{c}{d} = \frac{ad+bc}{bd}\) (b,d ≠ 0)
\(\frac{a}{b} * \frac{c}{d} = \frac{ac}{bd}\); \(\frac{\frac{a}{b}}{\frac{c}{d}} = \frac{ad}{bc}\) (b,d,c ≠ 0)
Vorzeichenregeln:
\( (-a)b = a(-b) = -ab \)
\( (-a)*(-b) = ab \)
Erweitern/Kürzen
\( \frac{ac}{bc} = \frac{a}{b} \) (b,c ≠ 0)
Nullprodukt
ab = 0 ⇔
a = 0 oder b = 0
Ungleichungen Die reellen Zahlen sind ungeordnet, in dem Sinne,
dass es eine "<" (kleiner) Relation gibt,
die folgende Eigenschaften besitzt:
seien a,b,c ∈ ℝ
Trichotomie Es gilt genau eine der Eigenschaften
a < b, a = b, a > b
Transitivität a < b,  b < c,  a < c
Monotoniebezüglich a < b ⇒ ac < bc für 0 < c;
